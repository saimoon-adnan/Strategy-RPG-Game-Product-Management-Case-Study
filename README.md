# Strategy RPG Game Product Management Case Study

This repository contains a **Product Management case study** based on a job simulation from Electronic Arts hosted on Forage.

The goal of this project is to demonstrate a structured approach to **product analysis, KPI selection, and data-driven decision making** within the video game industry.

## Project Objectives

The main objectives of this project were to:

- Understand the **Key Performance Indicators (KPIs)** used in the gaming industry  
- Select the most relevant KPIs to evaluate specific business problems  
- Analyze player engagement, retention, and monetization metrics  
- Break down the steps required to create a **data-driven product presentation**

---

## Project Overview

In this simulation, the role of a **Product Manager** involved evaluating the performance of a **Strategy RPG mobile game**.

Key responsibilities included:

- Identifying performance metrics for the game  
- Selecting appropriate KPIs to measure success  
- Structuring a presentation to communicate insights and recommendations  

The project demonstrates how **data-driven insights guide product decisions in the gaming industry**.

---

## Key Performance Indicators (KPIs)

Several KPIs were analyzed to evaluate the game’s performance.

### Engagement Metrics
- Daily Active Users (DAU)  
- Monthly Active Users (MAU)  
- Average Session Time  

These metrics measure how frequently players interact with the game.

---

### Retention Metrics
- Day 1 Retention  
- Day 7 Retention  
- Day 30 Retention  

Retention metrics indicate how effectively the game keeps players engaged over time.

---

### Monetization Metrics
- Average Revenue Per User (ARPU)  
- Conversion Rate (Free → Paying Users)  
- Lifetime Value (LTV)  

These metrics evaluate how effectively the game generates revenue.

---

## Product Management Approach

The analysis followed a structured product management process:

1. Understanding the product and business model  
2. Identifying the key business problem  
3. Selecting the most relevant KPIs  
4. Analyzing data and extracting insights  
5. Developing strategic recommendations  
6. Creating a clear and effective presentation plan
